# 🏥 MedCare – Doctor Appointment Booking Website

## 📌 Project Overview

MedCare is a responsive healthcare website that allows patients to book doctor appointments online. The website provides an easy-to-use interface for viewing healthcare services, finding doctors, and scheduling appointments conveniently.

## 🚀 Features

* Responsive and modern user interface
* Doctor appointment booking form
* Healthcare services section
* Doctor information display
* Contact section
* Mobile-friendly design
* Easy navigation

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```text
MedCare/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📋 Functionalities

### Home Page

Provides an introduction to MedCare and healthcare services.

### Doctor Appointment Booking

Patients can:

* Enter personal details
* Select a doctor
* Choose an appointment date
* Book appointments online

### Services Section

Displays available medical services and specialties.

### Contact Section

Provides contact details for patients.

## 💡 Advantages

* Saves time by booking appointments online
* User-friendly interface
* Accessible from desktop and mobile devices
* Improves patient convenience

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

No additional installation is required.
